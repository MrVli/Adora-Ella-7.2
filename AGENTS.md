# ADORA Shopify Theme Instructions

You are working on the ADORA Shopify theme.

## Brand Direction
ADORA is a premium funnel-first skincare brand.
The visual style must be elegant, feminine, calm, modern, clean, high-end, and conversion-oriented.
Avoid cheap dropshipping aesthetics, clutter, aggressive sale visuals, childish design, generic beauty clichés, and unnecessary animations.

## Shopify Rules
- This is a Shopify Liquid theme.
- Do not break existing Shopify theme structure.
- Keep sections modular and reusable.
- Prefer clean Liquid, CSS, and minimal JavaScript.
- Do not add external libraries unless explicitly requested.
- Prioritize mobile-first UX and fast loading.
- Avoid editing live production assumptions unless instructed.
- Never hardcode product IDs unless instructed.
- Use schema settings where possible so sections remain editable in Shopify Theme Editor.

## Conversion Priorities
1. Mobile UX
2. Loading speed
3. Clear CTA visibility
4. Premium perceived value
5. Trust
6. Funnel/PDP clarity
7. Maintainability

## Code Style
- Keep CSS scoped to the section where possible.
- Use semantic class names with `adora-` prefix for custom sections.
- Avoid unnecessary JavaScript.
- Comment only where helpful.
- Preserve existing theme functionality unless the task says to remove it.

## Testing Expectations
After changes, summarize:
- files changed
- what was added
- what could break
- how to test in Shopify preview
