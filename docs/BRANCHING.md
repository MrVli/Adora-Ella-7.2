# Branching Guide

Recommended workflow:

1. Keep `main` as the stable theme state.
2. Create one feature branch per meaningful change.
3. Use pull requests before merging larger changes.
4. Test every change in Shopify Theme Preview before publishing.

Example feature branches:

- `feature/adora-ritual-hero`
- `feature/adora-ritual-steps`
- `feature/adora-offer-stack`
- `feature/adora-pdp-support-sections`

Avoid working directly on the live theme without GitHub review.
