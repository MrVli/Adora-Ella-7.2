# ADORA Theme Rules

## Brand feel
ADORA must feel premium, elegant, clean, feminine, calm, modern, trustworthy, and conversion-focused.

## Avoid
- cheap dropshipping aesthetics
- cluttered layouts
- fake luxury design
- aggressive sale visuals
- generic beauty copy
- unsupported skincare claims
- unnecessary animations
- unnecessary JavaScript
- unscoped CSS

## Homepage structure target
1. Premium Glass Skin Reset Ritual hero
2. Problem/solution framing
3. Ritual explanation: Cleanse, Hydrate, Nourish
4. Bundle/offer stack
5. Ingredients/trust
6. Routine steps
7. Objection handling
8. Reviews/UGC
9. Final CTA

## PDP structure target
1. Strong product and ritual positioning
2. Clear product media
3. Visible CTA
4. Bundle/value framing
5. Routine explanation
6. Trust/ingredients
7. Objection handling
8. Reviews
9. Guarantee or risk reversal

## Technical direction
Use custom ADORA sections instead of over-editing Ella demo blocks. Keep changes modular, small, and reversible.
