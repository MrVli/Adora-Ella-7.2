# Shopify Preview Checklist

Run this checklist before publishing a theme update.

## Global
- [ ] Homepage loads
- [ ] Product page loads
- [ ] Collection page loads
- [ ] Cart page or cart drawer works
- [ ] Theme Editor opens without schema errors
- [ ] Navigation and footer work

## Mobile
- [ ] Hero does not crop important content badly
- [ ] CTA is visible and easy to tap
- [ ] Text hierarchy is readable
- [ ] No horizontal scrolling
- [ ] Sticky elements do not block checkout/cart actions

## PDP
- [ ] Product media loads
- [ ] Variant selectors work
- [ ] Quantity selector works
- [ ] Add to cart works
- [ ] Price updates correctly
- [ ] Trust/offer sections do not push CTA too far down without reason

## Performance
- [ ] No oversized images
- [ ] No unnecessary autoplay video
- [ ] No obvious console errors
- [ ] No layout shifts that hurt the first screen

## Brand quality
- [ ] Layout feels premium, clean, calm, and trustworthy
- [ ] No generic demo copy remains
- [ ] No aggressive fake-sale feeling
- [ ] Copy is specific and low-risk
